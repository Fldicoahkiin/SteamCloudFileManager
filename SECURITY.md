# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| < 1.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through [GitHub Security Advisories](https://github.com/Fldicoahkiin/SteamCloudFileManager/security/advisories/new).

We aim to respond to security reports within 48 hours and will work with you to assess and address the issue promptly.
